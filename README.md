# gama
Good Day!

Gama is a simple project build in C to easily build games(or apps) using opengl in C.
Is is basically made to work with windows, but I may extend, or not.

## sample gama program

```c
#include "gama.h"

void init() {
	// create shapes&physics
}
void update() {
	// update geometry and physics
}
void render() {
	// render the shapes
}
```
